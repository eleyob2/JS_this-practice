Overview

In JavaScript, this refers to the object that is currently executing a function. Inside an object method, this points to that object. Inside a class, this points to the instance being created or used.

Complete all five tasks in script.js. Every task is isolated — fill in the // TODO sections using this to access properties and methods. Click Run Tests to check your work.

TASK 01 Object Method — Basic Property Access

A person object has name and age properties. Complete the greet() method so it uses this to return an introduction string.

Requirements
  • Use this.name and this.age inside the method
  • Return the string "Hi, I'm [name] and I'm [age] years old."
  • Do not hard-code the values — use this
TASK 02 Object Method — Calling Other Methods with this

A counter object tracks a count. Complete increment(), decrement(), reset(), and getCount() using this to read and update the shared count property.

Requirements
  • All four methods must use this.count
  • increment adds the given amount; decrement subtracts it
  • reset() sets the count back to 0
  • getCount() returns the current value
TASK 03 Constructor Function — Building with this

Complete the Animal constructor function so it uses this to store the name and sound arguments as properties. Then complete the speak() prototype method.

Requirements
  • Use this.name and this.sound in the constructor
  • speak() must return "[name] says [sound]!" using this
  • Create instances with new Animal(...) to test it
TASK 04 Class — Properties, Methods, and this

Complete the Rectangle class. The constructor receives width and height. Complete area(), perimeter(), and describe() — the last one must call the other two using this.

Requirements
  • Store width and height on this in the constructor
  • area() returns width × height
  • perimeter() returns 2 × (width + height)
  • describe() calls this.area() and this.perimeter() and returns a summary string
TASK 05 Class — Method Chaining with this

Complete the ShoppingCart class. Returning this from a method allows method chaining — calling multiple methods in a row on the same object. Complete addItem(), applyDiscount(), and getTotal().

Requirements
  • addItem(name, price) pushes an item object into this.items and returns this
  • applyDiscount(percent) stores the discount on this and returns this
  • getTotal() sums all item prices, applies the discount percentage, and returns the final number
  • Chaining must work: cart.addItem(...).addItem(...).applyDiscount(10).getTotal()
